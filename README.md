# docker
## 安装docker
- 执行下面指令（仅支持ubuntu和centos，请确保操作系统内核版本符合docker官方要求）
```
curl -L github.com/OracleGao/docker/raw/master/bin/docker-install.sh | bash
```

## 安装docker-compose
- 执行下面指令
```
curl -L github.com/OracleGao/docker/raw/master/bin/docker-compose-install.sh | bash
```
