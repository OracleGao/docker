# Redis

## 环境变量说明
- REDIS_IMAGE_TAG docker镜像tag，默认5-alpine，详见[redis docker hub](https://hub.docker.com/_/redis)
- REDIS_PORT 端口，默认6379
- REDIS_CONF_PATH 配置文件路径，默认./conf/redis，配置文件名固定为redis.conf
- REDIS_DATA_PATH 数据文件路径，默认./data/redis
